# Success Salad 🥗 | سلطة النجاح

A clean, bilingual iOS productivity timer that helps you build a focused work habit.
Log sessions, earn vegetable-themed star ratings, and track your progress over time.

## Features
- ⏱ Session timer with vegetable themes (each vegetable = a different duration)
- ⭐ Star rating system based on session length
- 📊 Statistics — weekly progress chart, streaks, favorites
- 📝 Session log with full history
- 🌐 Full Arabic & English support (RTL/LTR)
- 🔔 Customizable alarm volume
- 🌙 Dark mode

## Built With
- SwiftUI
- SwiftData
- Swift 6

iOS 17+ | iPhone & iPad
